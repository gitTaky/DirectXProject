# 编译
DirectX 11 Engine VS2017\DirectX 11 Engine VS2017 目录下 Libs.zip文件解压并覆盖源文件
编译project

# 运行
若运行目录下不存在Data文件夹，请将DirectX 11 Engine VS2017\DirectX 11 Engine VS2017\Data 复制到运行目录
运行DirectX 11 Engine VS2017.exe

# 操作说明
## 视角切换
1. 自由视角
2. 第一视角
3. 第三视角

键盘：C 键进行切换

## 移动
键盘： WASD 前后左右

* 自由视角  
键盘：z space 上下
鼠标：右键拖动 移动视角

## 控制面板
左右拖动对应参数进行调节
